Overview
--------

The geniza project django application is broken up into two primary applications, with corresponding models in the database. The :mod:`geniza.corpus` application handles :class:`~geniza.corpus.models.Documents` and :class:`~geniza.corpus.models.Fragments` along with related models, functionality and views. The :mod:`geniza.footnotes` application provides functionality and models for Scholarship Records â€” :class:`~geniza.corpus.models.Source` records, and :class:`~geniza.footnotes.models.Footnote` objects connecting sources to documents.

More detailed `database documentation <https://dbdocs.io/princetoncdh/geniza>`_ is is autogenerated and hosted via dbdocs.io.
